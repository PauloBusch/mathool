# Mathool Client

## Framework e Cli

| Nome | Versão
|---| ---|
| vue.js | 3.1.2 |
| @vue/cli | 4.5.13 |

### Instalando Framework e Cli

```
npm install -g @vue/cli @vue/cli-service-global
```

### Gerênciando Projeto Com GUI do Vue-Cli

Você também pode criar e gerenciar projetos usando uma interface gráfica com o comando:

```
vue ui
```

Execute server, testes, lint, dentre outros com a GUI do vue-cli

## Dependências

| Dependência | Versão
|---| ---|
| eslint | 6.8.0 |
| core | 3.15.2 |
| vue-router | 4.0.10 |
| vuex | 4.0.2 |
| vuetify | 2.5.6 |

## Instalando Dependências

```
npm install
```

## Executando Server

```
npm run serve
```

## Build da Aplicação

```
npm run build
```

## Lint

```
npm run lint
```
